Site do Grupo de Usuários WordPress Brasil

### Instalar dependências ###

    $ bundle install

### Comandos rake ###

##### Assistir o projeto #####

    $ rake watch

##### Preview do projeto #####

    $ rake preview

##### Fazer build #####

    $ rake build

##### Fazer deploy #####

    $ rake deploy

##### Publicar um post #####

    $ rake new_post["Post Title","categories"]

##### Publicar uma página #####

    $ rake new_page["Page Title","layout","path/to/file"]
